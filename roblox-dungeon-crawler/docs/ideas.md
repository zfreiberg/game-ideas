# Ideas & Brainstorming

Raw ideas — anything goes here first. Not all of these will make it into the game.

---

## Core Game Loop
- [ ] Run dungeons, get gear, unlock next dungeon — WoW-style progression
- [ ] Gear should visibly improve your character (stats + cosmetic)
- [ ] Each dungeon clearly harder but more rewarding than the last

## Hub World
- [ ] Central hub world where players spawn and hang out between runs
- [ ] Multiple doors visible from hub, each leading to a different dungeon
- [ ] Doors have visual indicators of difficulty (color coding, lock icons, level req)
- [ ] Doors locked until player meets requirements (gear score, level, etc.)
- [ ] Hub world is a social space — players can see each other's gear

## Dungeons (Initial / Bare-Bones Pass)
- [ ] Enemies = red cubes (placeholder), they drop EXP on death
- [ ] When all enemies in dungeon are defeated → popup window appears
- [ ] Popup offers: "Return to Hub" button
- [ ] Dungeons have increasing difficulty tiers (Dungeon 1, 2, 3...)
- [ ] Clear time or score tracked per run (future leaderboard potential)

## Combat & Enemies
- [ ] Placeholder red cube enemies to start
- [ ] Enemies patrol or stand still (simple AI)
- [ ] Click-to-attack or proximity damage to start
- [ ] Boss enemy at end of dungeon (bigger cube?)

## Gear & Progression
- [ ] Gear drops from enemies or chests in dungeon
- [ ] Gear has stats (damage, health, speed)
- [ ] Gear score determines which dungeons you can enter
- [ ] Inventory / equipment UI

## Monetization (Future)
- [ ] Cosmetic-only — no pay to win
- [ ] Cosmetic gear skins, hub world pets, emotes

## Viral / Social Hooks (Future)
- [ ] Leaderboard for fastest clears
- [ ] Party/co-op dungeon runs
- [ ] Daily challenge dungeon
